# Task 1 - Multiplication function.  
In this project, a small C++ calculator is implemented, capable of performing multiplication operations with integer numbers. CMake was used for building, and Catch2 for unit tests.  
# Build
### Configuration (setup) of the project  
```bash
cmake -S . -B Build
```
### Build the project
```bash
cmake --build build
```
# Run  
```bash
./build/Project_1
```
# Unit tests  
```bash
./build/tests
```
